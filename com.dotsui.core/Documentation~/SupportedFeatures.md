# Supported features

## Canvas

### Render modes:
* Screen space camera

### Scaling modes:
* Constant pixel size
* Constant physical size

## Controls:

* Text (TMP)
* Image
* Button
* ScrollView
* InputField (requires rework)

## Low-level input events:

* Down
* Up
* Click
* Enter
* Exit
* Selected
* Deselected
* BeginDrag
* Drag
* EndDrag
* Drop

## High-level events

* Button click
* InputField OnEndEdit
* InputField OnReturn
